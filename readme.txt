= Privacy Notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* write any user personal data to the database;
* use cookies.

== Installation ==

1. Upload the entire `modpinn` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You may find 'Modpinn' menu in your WordPress admin panel.

Have a look at the [plugin web site](https://pressfinesse.com/).
